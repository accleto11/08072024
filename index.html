How to clone a github repository

  1. Cloning the Repository

    git clone

  2. Creating a New Branch

    cd git checkout -b

  3. Making Changes

    git add ... or git add .

  4. Committing Changes

    git commit -m "Your descriptive commit message"

  5. Pushing Changes to GitHub

    git push origin

  6. Alternative: Creating a Pull Request via GitHub CLI

    gh pr create --base --head --title "Your PR Title" --body "Description of your PR"

Creating a Pull Request

To create a pull request (PR), you need to use the GitHub website. Here's how:

  1. Navigate to your repository on GitHub.
  2. Switch to your new branch by clicking the branch dropdown and selecting it.
  3. Click on the "Compare & pull request" button.
  4. Provide a title and description for your pull request.
  5. Review the changes to ensure everything looks correct.
  6. Click on the "Create pull request" button.
